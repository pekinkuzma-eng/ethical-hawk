# Ethical Hawk

Ethical Hawk — система обнаружения и предотвращения атак на веб-приложение.

## Возможности

- Анализ сетевого трафика
- Обнаружение SQL Injection
- Обнаружение XSS
- Обнаружение Brute Force
- Логирование атак
- Сохранение в SQLite
- GUI интерфейс
- Автоматическая блокировка IP

## Технологии

- Python
- Scapy
- SQLite
- Tkinter
- Threading
- iptables

```bash
sudo python3 main.py
